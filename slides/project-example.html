Typical project scaffold

/project-name
  ↳ package.json      // Project manifest
  ↳ gulpfile.js       // Gulp config file and/or
  ↳ Gruntfile.js      // Grunt config file
  ↳ /node-modules     // Node packages (autogenerated by NPM)
  ↳ /[src]            // Some source files somewhere
  ↳ /[build]          // Some place you want save the results
